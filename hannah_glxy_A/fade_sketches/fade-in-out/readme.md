## Star Color to Known Stellar Classification Ratios. 
### This Project uses inverse values (ex. least frequent star color in space replaced with most frequent)
- blue.............. 1/70 (actual/inversed for project)
- blue white........2/11
- white.............3/8
- yellow white......5/5
- yellow............8/3
- light orange......11/2
- orange red........70/1
